## WooCommerce POS WCPOS Sumup Tap To Pay payment Gateway

Sumup Tap To Pay payment

### Instructions

1. Check [releases](https://github.com/TheStingPilot/sumup-tap-to-pay-gateway/releases) for the latest version of the plugin.
2. Download the **woocommerce-pos-wcpos-sumup-tap-to-pay-payment-gateway.zip** file.
3. Install & activate the plugin via `WP Admin > Plugins > Add New > Upload Plugin`.
4. Go to `WP Admin > WooCommerce POS > Settings > Checkout > enable` the WooCommerce POS WCPOS Sumup Tap To Pay payment Gateway.

### Development

To customize this gateway, edit the `wcpos-wcpos-sumup-tap-to-pay-payment.php` file and implement your payment processing logic in the `process_payment()` method.
